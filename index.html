Blazingly Fast

Written entirely in Rust with an 8MB atomic frame buffer. It opens instantly and never tears, even on massive codebases.

Smart Autocomplete

Built-in Language Server Protocol (LSP) and Tree-sitter gives you VS Code level autocomplete directly in your terminal.

12 Built-in Themes

Switch between Catppuccin, TokyoNight, Kanagawa, Gruvbox, and more instantly. No extra configuration required.

Install Rux

You need Rust and Cargo installed on your system.

git clone https://github.com/your-username/rux.git
cd rux
cargo install --path .
