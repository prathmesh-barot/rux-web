Rux Documentation

Welcome to the official documentation for Rux Editor. Here you will find everything from basic installation to advanced LSP configuration.

Architecture Overview

Rux is designed around a single, highly-optimized atomic frame buffer. It skips all the heavy abstraction layers found in modern electron apps to give you unparalleled speed in your terminal.

// Fast, simple, reliable.
pub struct RuxConfig {
    pub editor: EditorConfig,
    pub lsp: LspConfig,
}

Installation

Currently, Rux is distributed as source. You will need the latest stable release of Rust to compile it on your machine.

cargo install --git https://github.com/your-username/rux.git

Quick Start

Once installed, you can simply run `rux` in your terminal to open an empty buffer, or pass a file or directory directly.

rux ./src/main.rs
