Rux v0.5.0: The Theme Engine Update

We are incredibly excited to announce the release of Rux v0.5.0! This release focuses heavily on UI, UX, and bringing beautiful default aesthetics without relying on plugins.

What's new?

  • 12 Built-in Themes: We've baked in the most popular terminal themes directly into the binary. From Catppuccin and TokyoNight, to Gruvbox and Kanagawa. Switch instantly using :theme <name>.
  • Atomic Redraw: The terminal UI buffer has been rewritten to guarantee 0% tearing when scrolling fast through large files.
  • Split Panes: Use Ctrl+W to manage horizontal and vertical splits effortlessly.
March 10, 2026 • Release

Rux v0.4.0: LSP Integration

Rux is no longer just a text editor—it's an IDE inside your terminal. With v0.4.0, we have integrated a full LSP client natively.

  • Zero-config support for `rust-analyzer`, `gopls`, and `pyright`.
  • Inline diagnostic rendering (errors/warnings).
  • Fast, responsive dropdowns for auto-completion.
